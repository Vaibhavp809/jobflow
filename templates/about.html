About JobFlow
A structured job application management system built using Django and Django REST Framework.

Project Purpose

JobFlow helps users track job applications, monitor interview stages, and visualize progress using interactive analytics.

Core Features

• Dashboard Overview
• Full CRUD Operations
• Status Filtering
• Interactive Analytics
• REST API Support

Technology Stack

Django
Django REST Framework
SQLite
Chart.js
HTML5
CSS3

Architecture

The system follows a modular Django structure, separating API logic from template-based UI, ensuring maintainability and scalability.
